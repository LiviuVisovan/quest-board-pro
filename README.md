# Quest Board Pro

A simple full-stack quest board app built with Next.js and Prisma.

![Quest Board Pro Screenshot](./questboard.png)

## Features

- Create, edit, and delete quests
- Priority and status tracking
- Persistent database using Prisma
- Clean UI with loading and error states

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Prisma
- SQLite

## Getting Started

```bash
npm install
npx prisma migrate dev
npm run dev
```
